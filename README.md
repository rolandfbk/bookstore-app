## Bookstore

> This is a project built with React. It is a single page application that displays a list of books, and allows users to add and delete books.

## Setup
>- Git clone `https://github.com/rolandfbk/bookstore-app.git`

## Usage
>- npm install
>- npm run build
>- npm start

## Test
>- npm test
